#  PEAK  **Proactive Engine for Assessment & Knowledge**

An AI-powered Offensive Security Orchestration platform featuring active reconnaissance, MobSF integration, and LLM-driven attack planning.

## Features
- Active Profiling (Headers, Git, Env probing)
- Mobile App Scanning (APK/IPA Static Analysis)
- AI Attack Planner (Phi-3 / Ollama)
- CSV Reporting

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Rename `config.example.py` to `config.py` and add your keys.
3. Run: `python app.py`
